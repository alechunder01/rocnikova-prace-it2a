# Ročníková práce

Ročníková práce druhého ročníku žáka Alexandre Nicolas.

## Gameplay

Vzlétnout ze Země a dostat se k měsící.

Hráč během cesty plní náhodné úkoly:

- odlet ze Země
  - nastavit lod
  - odhodit boostery
  - atd...
- experimenty
- roztáhnout solární panely
- opravit něco
- opravit kurz
- připravit jídlo
- atd...
